Урок 1. Ёмкости

Цель урока: научиться распознавать ёмкости в окружающем мире.

Рассмотрим для начала вспомогательные определения.


Ёмкость без элементов называется пустой ёмкостью.

Примеры ёмкостей:
  • Замок - вместилище для принцессы и дракона.
  • Шапка - место для головы.
  • Стол - место для вещей.
  • Банка - ёмкость для мёда.
  • Автомобиль вмещает пятерых пассажиров.
  • Головы людей вмещают идеи.
  • Свято место пусто не бывает.
  • Безделье - пустое времяпрепровождение.
  • Планета Земля - ёмкость для атмосферы.
  • Море - ёмкость для рыб.
  • Космос - ёмкость для звёзд.
  • Атом - ёмкость для электронов
  • Человек - ёмкость для одежды.
  • Любовь - ёмкость для влюблённых.


Устройство - это ёмкость, связи между составляющими которой позволяют ей иметь цель существования, то есть выполнять функцию, решая задачи.

Примеры устройств:
  • Рация военного является электротехническим изделием и предназначена для поддержания связи между сотрудниками воинских подразделений.
  • Ветряная мельница представляет собой механизм, служащий для перемалывания зерна в муку
  • Почва является естественной средой обитания многочисленных организмов


Бесполезные для субъекта вещи и явления, не имеющие своего предназначения, то есть не могущие нести никакой полезной нагрузки, устройствами не являются, но со временем могут переходить в категорию устройств по мере накопления знаний о них. Например, русский язык является стройной, органичной системой обмена информацией для владеющих им, но русская речь окажется нечленораздельной тарабарщиной для иноязычных.

Задание на дом:
  1. Придумайте самостоятельно пять примеров ёмкостей, которые не встречаются в жизни, и обсудите с другими учащимися.
  2. Приведите десять примеров ёмкостей, которые встречаются в жизни, и обсудите с другими учащимися.
  3. Обсудите с другими учащимися, почему ёмкий и вместительный являются синонимами.
  4. Выпишите из толкового словаря Ожегова последнего издания определения места, пространства и содержимого и объясните, как Вы их понимаете.

Вопросы для обсуждения по теме урока:
  1. Как Вы думаете, может ли ёмкость содержать бесконечное число элементов и почему?
  2. Может ли общество людей быть ёмкостью и почему? Если да, то что она может вмещать, и где находятся границы такой ёмкости?
  3. Какие действия можно совершать над ёмкостями? А над их содержимым?
